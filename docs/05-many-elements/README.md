# Тема 5: Робота з множинними елементами

## Огляд теми

Ця тема охоплює методи для роботи з колекціями елементів: ітерація, вибір крайніх елементів та навігація по сусідніх елементах.

## Основні методи

### 1. each() - ітерація через елементи

`each()` виконує функцію для кожного елемента в колекції.

### 2. first() та last() - крайні елементи

`first()` та `last()` вибирають перший та останній елемент.

### 3. Навігаційні ланцюжки

- `nextAll()` - всі наступні сусідні
- `prevAll()` - всі попередні сусідні
- `nextUntil()` - наступні до певного
- `prevUntil()` - попередні до певного
- `parents()` - всі батьківські елементи
