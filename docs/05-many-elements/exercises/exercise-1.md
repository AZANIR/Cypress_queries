# Вправа 1: Робота з множинними елементами

## Завдання

Створіть тест, який:

1. Додає 3 задачі
2. Використовує `each()` для ітерації через всі задачі
3. Для кожної задачі перевіряє, що вона видима
4. Використовує `first()` та `last()` для вибору крайніх елементів

## Очікуваний результат

Тест має успішно ітерувати через всі елементи.
