# Тема 4: Аліаси в Cypress

## Огляд теми

Аліаси дозволяють надавати імена елементам для подальшого використання. Це покращує читабельність та підтримку тестів.

## Створення та використання аліасів

### Створення аліасу:

```javascript
cy.get('.element').as('elementName')
```

### Використання аліасу:

```javascript
cy.get('@elementName')
```

## Переваги аліасів

- Покращена читабельність
- Легша підтримка
- Менше дублювання коду
- Менша залежність від селекторів
