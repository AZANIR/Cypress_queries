# Вправа 1: Створення та використання аліасів

## Завдання

Створіть тест, який:

1. Створює аліас для поля вводу `.new-todo`
2. Використовує аліас для введення тексту
3. Використовує аліас для перевірки значення
4. Використовує аліас для очищення поля

## Очікуваний результат

Тест має успішно використати аліас у різних місцях.
