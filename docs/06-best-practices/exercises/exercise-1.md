# Вправа 1: Best Practices

## Завдання

Створіть тест, який:

1. Використовує аліаси для основних елементів
2. Використовує `within()` для обмеження контексту
3. Використовує семантичні селектори
4. Уникає селекторів, що залежать від позиції

## Очікуваний результат

Тест має демонструвати best practices для створення стабільних тестів.
