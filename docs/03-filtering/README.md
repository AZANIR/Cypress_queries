# Тема 3: Фільтрація елементів у Cypress

## Огляд теми

Фільтрація дозволяє вибирати підмножину елементів з колекції за певними умовами. Це потужний інструмент для роботи з множинами елементів.

## Основні методи фільтрації

### 1. filter() - фільтрація за умовою

`filter()` залишає тільки елементи, що відповідають умові.

**Підтримує:**
- CSS селектори
- Псевдокласи
- Функції для складних умов

### 2. not() - виключення елементів

`not()` виключає елементи, що відповідають умові.

**Коли використовувати:**
- Для виключення небажаних елементів
- Для вибору активних елементів (не виконаних)
- Для спрощення селекторів
