# Вправа 1: Фільтрація елементів

## Завдання

Створіть тест, який:

1. Додає 3 задачі
2. Відмічає одну задачу як виконану
3. Використовує `filter()` для пошуку виконаних задач
4. Використовує `filter()` для пошуку активних задач
5. Перевіряє кількість кожної категорії

## Очікуваний результат

Тест має успішно відфільтрувати задачі за станом.
