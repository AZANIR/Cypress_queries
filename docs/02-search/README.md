# Тема 2: Пошук елементів у Cypress

## Огляд теми

Ця тема охоплює всі основні методи пошуку елементів у Cypress. Ви дізнаєтесь про різні способи знаходження елементів на сторінці та навігації по DOM дереву.

## Основні методи пошуку

### 1. cy.get() - основний метод пошуку

`cy.get()` - це основа пошуку елементів у Cypress. Він підтримує CSS селектори та автоматично чекає на появу елементів.

**Особливості:**
- Автоматичне очікування (до 4 секунд за замовчуванням)
- Підтримка CSS селекторів
- Може знайти кілька елементів одночасно
- Повертає jQuery об'єкт

### 2. find() - пошук всередині елемента

`find()` знаходить нащадкові елементи всередині вже знайденого елемента.

**Коли використовувати:**
- Коли потрібно знайти елемент всередині іншого
- Для покращення читабельності коду
- Для уникнення складних селекторів

### 3. contains() - пошук за текстом

`contains()` знаходить елемент за текстовим вмістом (часткове співпадіння).

**Особливості:**
- Повертає тільки перший елемент
- Чутливий до регістру
- Може працювати як окрема команда або метод ланцюжка

### 4. Навігаційні методи

- `children()` - дочірні елементи
- `parent()` - батьківський елемент
- `closest()` - найближчий предок
- `siblings()` - сусідні елементи
- `prev()` - попередній сусідній елемент

### 5. Спеціальні методи

- `focused()` - елемент у фокусі
- `invoke()` - виклик методів
- `its()` - отримання властивостей
- `title()` - заголовок сторінки
