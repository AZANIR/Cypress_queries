# Вправа 1: Базовий пошук елементів

## Завдання

Створіть тест, який:

1. Відкриває сторінку TodoMVC
2. Використовує `cy.get()` для пошуку поля вводу
3. Використовує `cy.get()` для пошуку заголовка
4. Перевіряє, що обидва елементи існують та видимі

## Очікуваний результат

Тест має успішно знайти та перевірити елементи.
