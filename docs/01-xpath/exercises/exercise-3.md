# Вправа 3: XPath селектори

## Завдання

Створіть тест, який:

1. Відкриває сторінку TodoMVC
2. Знаходить заголовок через XPath: `//h1`
3. Знаходить поле вводу через XPath: `//input[@type="text"]`
4. Додає задачу та знаходить її через XPath: `//li[contains(text(), "Задача")]`

## Підказки

- Переконайтеся, що `cypress-xpath` встановлено та імпортовано
- Використовуйте `cy.xpath()` замість `cy.get()`
- Використовуйте `contains()` для пошуку за текстом

## Очікуваний результат

Тест має успішно знайти елементи через XPath селектори.
