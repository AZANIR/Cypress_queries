# Вправа 1: CSS Селектори

## Завдання

Створіть тест, який:

1. Відкриває сторінку TodoMVC
2. Знаходить поле вводу за класом `.new-todo`
3. Знаходить заголовок `h1` та перевіряє, що він містить текст "todos"
4. Знаходить список задач `.todo-list` та перевіряє, що він існує

## Підказки

- Використовуйте `cy.visit('/')` для відкриття сторінки
- Використовуйте `cy.get('.class-name')` для пошуку за класом
- Використовуйте `.should('contain', 'text')` для перевірки тексту
- Використовуйте `.should('exist')` для перевірки наявності

## Очікуваний результат

Тест має успішно пройти, знайшовши всі елементи.
