# Вправа 2: Комбіновані CSS селектори

## Завдання

Створіть тест, який:

1. Додає 3 задачі через поле вводу
2. Знаходить всі задачі через комбінований селектор `.todo-list li`
3. Перевіряє, що знайдено рівно 3 задачі
4. Знаходить першу задачу через `:first-child`
5. Знаходить останню задачу через `:last-child`

## Підказки

- Використовуйте `cy.get('.new-todo').type('Задача{enter}')` для додавання задач
- Використовуйте `.should('have.length', 3)` для перевірки кількості
- Використовуйте псевдокласи `:first-child` та `:last-child`

## Очікуваний результат

Тест має успішно знайти всі задачі та перевірити їх кількість.
